## 2. 项目目标

开发一个命令行工具，用于遍历目录、解析 Excel 文件（.xlsx/.xls/.xlsb/.ods），并将数据转换为 JSON 配置格式。

核心特性：
- 目录递归遍历（可配置开关）
- Excel 内嵌元数据约定：每个 sheet 自描述（configName、description、字段定义）
- `id` 列强制：Col A 统一为唯一 u64，`//` 前缀为注释行
- 通过 TOML 配置文件驱动字段映射（列重命名、排除、嵌套路径）
- 单行 JSON 摘要输出到 stdout，供父进程解析
- stderr tracing 诊断日志
- CLI 参数覆盖配置文件
- stdin 管道接收 `outputDir`
- 逐文件/逐 sheet 错误容错
- camelCase configName → kebab-case 输出文件名自动转换
