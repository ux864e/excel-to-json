## 2. 项目目标

开发一个命令行工具，用于遍历目录、解析 Excel 文件（.xlsx/.xls/.xlsb/.ods/.csv），并将数据转换为 JSON 格式。

核心特性：
- 目录递归遍历
- 通过 TOML 配置文件驱动字段映射（列重命名、排除、嵌套路径）
- stdout 行协议 IPC，实时通知转换进度
- CLI 参数覆盖配置文件
