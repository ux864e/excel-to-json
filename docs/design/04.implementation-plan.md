# excel-to-json — 实施计划

## 1. 当前状态

**Phase 3: 稳定与完善** — 进行中 (2026-06-29)

所有核心功能已实现并测试通过（34 个测试，含 24 单元 + 10 集成）。

## 2. 已完成阶段

### Phase 0: 项目初始化 ✅ (2026-06-27)
- [x] Git 仓库初始化
- [x] Cargo 项目骨架
- [x] 源码模块框架（编译通过，测试全绿）
- [x] Agent 配置（CLAUDE.md, AGENTS.md, 8 个 instructions 文件）
- [x] CI/CD 配置
- [x] 设计文档

### Phase 1: 核心转换逻辑 ✅ (2026-06-29)
- [x] converter.rs 完整实现：calamine Excel 解析、`parse_sheet_with_meta` 元数据提取
- [x] `validate_config_name` 校验（`^[a-z][a-zA-Z0-9_]*[a-zA-Z0-9]$`）
- [x] `id` 列强制：u64 解析、`//` 注释跳过、重复检测
- [x] mapping.rs 完整实现：列重命名、排除、嵌套路径、cell-to-JSON
- [x] 行级别统计：inputRows/validRows/skippedRows/failedRows
- [x] 测试 Excel fixtures：sample.xlsx、multi-sheet.xlsx
- [x] 24 个单元测试

### Phase 2: 配置与 CLI ✅ (2026-06-29)
- [x] CLI + TOML 合并逻辑（CLI 覆盖配置文件）
- [x] stdin 管道支持（`read_stdin_config` 读取 `outputDir`）
- [x] 单行 JSON 摘要替代多行 IPC（`Summary`/`FileSummary`）
- [x] 逐文件/逐 sheet 错误容错（`ConversionSummary`）
- [x] 10 个集成测试

### Phase 3: 稳定与完善 🔄 (2026-06-29)
- [x] deploy.sh 部署脚本（自动检测平台，编译 release binary 更名为 config-importer）
- [x] setup-rust.sh 环境初始化脚本（幂等，USTC 镜像 fallback，PATH 自动配置）
- [x] camelCase configName → kebab-case 输出文件名（`camel_to_kebab`）
- [x] 注释列过滤（`comment_column_mask`，`//` / `#` 前缀 header）
- [x] 设计 docs、dev logs、README 同步
