| NFR-04 | 代码质量 | clippy -D warnings 零容忍 | ✅ |

## 3. 架构约束

- 单 crate，不拆 workspace
- 同步 I/O，不使用 async runtime
- `src/main.rs` 薄封装（≤10行），核心逻辑在 `src/lib.rs`
- 按 domain 组织模块，不按技术层
- 配置文件位于输入目录或通过 --config 指定

## 4. 数据流

```mermaid
flowchart LR
    CLI[CLI Args] -->|Override| CFG[Config]
    TOML[TOML File] -->|Base| CFG
    STDIN[stdin JSON] -->|outputDir| CFG
    CFG --> WALK[Directory Walker]
    WALK -->|ExcelFile[]| CONV[Converter]
    CONV -->|Sheet Parse| META[Metadata Extraction]
    META -->|Raw Rows| MAP[Field Mapping]
    MAP -->|Mapped JSON| OUT[Output Writer]
    OUT -->|.json files| DISK[Output Dir]
    OUT -->|Single-line JSON| STDOUT[stdout Summary]
```
